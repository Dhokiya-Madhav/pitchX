# 🏏 PitchReserve Frontend — Cricket Booking UI

This is the frontend application for PitchReserve, a cricket ground booking platform. It provides an intuitive UI for players to book slots and for owners to manage grounds and bookings.

---

## 🚀 Features

- Player & Owner authentication (JWT-based)
- Role-based UI (Player / Owner views)
- Ground discovery with city-based filtering
- Ground details and slot availability
- Slot booking and cancellation
- Owner dashboard (grounds, slots, bookings)
- Responsive UI with modern design

---

## 🛠️ Tech Stack

- React (Vite + TypeScript)
- TanStack Query (data fetching & caching)
- React Router (routing)
- Tailwind CSS (styling)
- shadcn-ui (UI components)
- Toasts & tooltips for UX
